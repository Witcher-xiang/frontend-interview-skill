# AI 应用开发面试模拟器

面向 **AI 应用开发岗位**求职者的模拟面试技能，涵盖 Agent、RAG、大模型应用、AI 编程等核心方向。

---

## 快速开始

在对话中直接说：

```
开始 AI 面试
```

技能将自动进入面试流程，引导你完成身份确认、风格选择，然后开始正式模拟面试。

---

## 核心功能

### 全流程模拟面试

1. **身份确认** — 实习生 / 应届生 / 社招生（1-3 年）
2. **风格选择** — 六种面试官风格可选
3. **项目深挖** — 结合你提供的项目经历深入提问
4. **技术考察** — LLM 原理、Agent、RAG、MCP、AI 编程等
5. **场景设计** — 客服 Agent、知识库问答、多模态等系统设计
6. **评分反馈** — 结构化评估报告 + 具体改进建议

### 六种面试官风格

| 风格 | 特点 | 推荐人群 |
|------|------|---------|
| 严厉拷打型 | 高压、快节奏、挑战每个回答 | 社招生 |
| 温和鼓励型 | 耐心、引导式、支持 | 实习生、应届生 |
| 专业高效型 | 结构清晰、时间精准、聚焦核心 | 目标明确者 |
| 深挖学术型 | 追求原理、讨论式、不求广但求深 | 对原理有追求者 |
| 工程实践型 | 生产视角、实战导向、不纸上谈兵 | 有项目经验者 |
| 综合平衡型 | 全面温和、不走极端 | 选择困难症 |

### 技术知识库覆盖

- **LLM 底层原理** — Token、上下文窗口、Temperature、幻觉成因与缓解
- **AI Agent** — Agent Loop、ReAct、Plan-and-Execute、Reflection、Multi-Agent
- **Context Engineering** — System Prompt 设计、短期/长期记忆、上下文优化
- **RAG 检索增强生成** — 分块策略、向量索引（HNSW/IVF）、召回优化、Re-ranker
- **MCP 协议与工具调用** — Function Calling、MCP 三大原语、Prompt Injection 防护
- **AI 编程实战** — Copilot/Cursor/Trae 使用技巧、适用场景与局限性
- **系统设计高频场景** — 客服 Agent、知识库问答、具身智能与多模态

## 反馈示例

面试结束后，你会收到一份结构化评估报告，包含：

- **综合评分** — 十分制，保留一位小数
- **分项评分** — AI 技术深度 / 项目经验 / 思维逻辑 / 表达能力 / AI 工具使用
- **具体改进建议** — 针对你的薄弱点，下一步具体可执行的学习计划
